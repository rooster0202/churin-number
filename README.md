# 自転車 駐輪番号 PWA

自転車の駐輪番号をすばやく保存して、精算機の前で大きく確認するための個人用PWAです。

## ローカル実行

```bash
python3 -m http.server 4175
```

ブラウザで `http://127.0.0.1:4175` を開きます。

## iPhoneで使う方法

PWAのインストールとオフラインキャッシュは、`https` のURLまたはローカル開発URLで動作します。実際に使う場合は、GitHub Pages、Netlify、Vercelなどの静的ホスティングに `bike-spot/` フォルダをアップロードして、Safariで開きます。

Safariで共有ボタンを押して `ホーム画面に追加` を選ぶと、アプリアイコンのように使えます。

## 機能

- 3〜4桁の番号入力
- 保存した番号を大きく表示
- 再読み込み後も番号を保持
- 番号コピー
- 精算完了時にリセット
- 基本的なオフラインキャッシュ
